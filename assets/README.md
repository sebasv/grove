# assets

Static images for the project README and other docs.

| File | Purpose | Notes |
|------|---------|-------|
| `screenshot.png` | Hero screenshot at the top of the README. | A real terminal capture of grove with at least one repo expanded, a terminal tab active, and a worktree showing badges. PNG, ~1200px wide. |
| `demo.gif` *(optional)* | Short animated demo. | Record with [`asciinema`](https://asciinema.org/) + `agg`, or a screen-recorder. Keep under ~5 MB. |

When updating, prefer overwriting these filenames so the README never has to change.
